# **App Name**: Veggie Box

## Core Features:

- User Accounts: User authentication and profile management.
- Box Selection: Display available CSA box options with detailed descriptions and pricing.
- Subscription Scheduling: Calendar view for selecting subscription start date.
- Payment Integration: Secure processing of subscription payments.
- Admin Management: Admin dashboard to manage subscriptions, user accounts, and box options.
- Reporting: Generate reports on active subscriptions, revenue, and popular box options.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to evoke freshness and nature.
- Background color: Light beige (#F5F5DC), a desaturated version of the primary color, for a warm, neutral backdrop.
- Accent color: Muted gold (#BDB76B) to highlight key actions and information, drawing from analogous colors but standing out from the primary.
- Body and headline font: 'PT Sans' for a friendly, accessible, modern look
- Use simple, line-based icons representing different vegetables and fruits.
- Clean, grid-based layout with ample white space to enhance readability.
- Subtle transitions and animations when navigating between sections or updating subscription details.
